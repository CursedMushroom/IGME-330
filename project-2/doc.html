Concept

  1. The user is tasked with observing and interacting woth mushrooms wothin there enviroment. The user can directly interact via dragging, freezing, or flinging them around. As well as being able to change the time of day via the sun/moon/time orb. the user can also change the weather, and the mushrooms will change there behavior, with find shelter or having there speed reduced. The user can summon a snail, the the mushrooms will try to avoid and stay away from. And while they wont seek them out, the user can spawn in flower crowns, the the mushrooms will put on when close enough to them.
